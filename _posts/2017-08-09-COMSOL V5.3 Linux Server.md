Failed systems
- CentOS 7.2 Xfce

Worked System
- Ubuntu 17.04 Xfce


Installation
```bash
mkdir /mnt/comsol
mount -o loop /home/centos/COMSOL53_dvd.iso /mnt/comsol
cd /usr/local/comsol53/server/bin/
./comsol server -silent -login never
```

Open web browser, enter address localhost:2036 (default server port)
